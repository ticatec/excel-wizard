## 🚀 Usage

### Method 1: Download from Example Pages

Each example page has a "Download Example Template" button that provides the appropriate Excel file.

### Method 2: Direct Download

Access files via:
```
http://localhost:5173/assets/examples/user-management-example.xlsx
http://localhost:5173/assets/examples/product-catalog-example.xlsx
http://localhost:5173/assets/examples/data-validation-example.xlsx
```

### Method 3: Regenerate Files

To regenerate these files with modified data:

```bash
npm run generate:examples
```

This runs the script at `scripts/generateExamples.ts` which creates fresh Excel files using ExcelJS.

1. **Start with valid data**: Use rows 1-10 to test successful uploads
2. **Test validation**: Use rows 11-14 to test error handling
